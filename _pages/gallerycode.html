Next steps

  1. Create a folder on your computer named where you will place the following files.
  2. Copy the code below and save it to a file named .txt in the root of the folder you just created.
  3. Add your app's icon at a size of exactly 75x75 and name it icon.jpeg.
  4. Add each of your app's screenshots to your folder with the file paths /screenshot##.jpeg. You may have up to 5 screenshots.
  5. At this point you should have the following directory structure: 
    /.txt
/icon.jpeg
//screenshot01.jpeg
//screenshot02.jpeg
//screenshot03.jpeg
//screenshot04.jpeg
//screenshot05.jpeg
  6. Archive the folder you created and email it to Jeff at jverkoey@gmail.com for review
